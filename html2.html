Sobre o site



Esse site foi desenvolvido para a disciplina de Desenvolvimento Web do curso Connect - Formação de programadores.
Para listar o todos os deputados disponiveis pela API Dados abertos acesse a aba "Lista de Deputados".
Na aba de "Deputados" você pode buscar os dados de um Deputado selecionando um dos itens do combo box.
Para listar os partidos e encontrar os membros do partido acesse a aba "Partidos"
Se quiser, também poderá consultar as despesas de cada deputado usando filtros de ano e mês, basta acessar a aba "Despesas"
